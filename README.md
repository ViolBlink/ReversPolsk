# Калькулятор на основе обратной польской нотации

Консольный калькулятор на C++, который вычисляет математические выражения, переводя их в обратную польскую нотацию (ОПН) с помощью алгоритма сортировочной станции.

## Поддерживаемые операции

| Оператор | Действие |
|----------|----------|
| `+` | Сложение |
| `-` | Вычитание |
| `*` | Умножение |
| `/` | Деление |
| `^` | Возведение в степень (правоассоциативный) |
| `( )` | Скобки (вложенные поддерживаются) |

Поддерживаются дробные числа (через `.`) и унарный минус.

## Сборка и запуск

### Требования
- CMake 3.15+
- Компилятор C++17 (GCC, Clang, MSVC)

### Сборка через CMake

```bash
cmake -S . -B build
cmake --build build
```

Бинарник будет помещён в папку `bin/`.

### Запуск

```bash
./bin/main
```

## Алгоритм

1. Входная строка очищается от пробелов (`DeleteSpase`).
2. Унарный минус преобразуется в `0-` (`HandleNegative`), например `-3+1` → `0-3+1`.
    - Из-за этого надо осторожно обращаться с пультипликативными операторами, так как -2^2 -> 0 - 2^2 = -4.
3. `ToPolsk()` переводит выражение из инфиксной записи в ОПН по алгоритму сортировочной станции:
   - числа сразу попадают в выходную очередь;
   - операторы помещаются в стек с учётом приоритета и ассоциативности (`^` — правоассоциативный);
   - скобки обрабатываются рекурсивно.
4. `FromPolsk()` вычисляет ОПН-очередь: числа кладутся в стек, при встрече оператора извлекаются два операнда, результат кладётся обратно.
