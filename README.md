# Калькулятор на основе обратной польской нотации

Консольный калькулятор на C++, который вычисляет математические выражения, переводя их в обратную польскую нотацию (ОПН).

## Поддерживаемые операции

| Оператор | Действие |
|----------|----------|
| `+` | Сложение |
| `-` | Вычитание |
| `*` | Умножение |
| `/` | Деление |
| `^` | Возведение в степень |
| `( )` | Скобки (вложенные поддерживаются) |

## Сборка и запуск

### Требования
- Компилятор C++11 или новее (GCC, Clang, MSVC)

### Сборка через g++
```bash
g++ main.cpp Functions.cpp -o calculator
```

### Запуск
```bash
./calculator
```

## Использование

```
Write your expression, to exit, write esc
Write hear ----> 3 + 4 * 2
11
Write hear ----> (3 + 4) * 2
14
Write hear ----> 2 ^ 10
1024
Write hear ----> esc
```

Для выхода введите `esc`.

## Алгоритм

1. Входная строка очищается от пробелов.
2. `ToPolsk()` переводит выражение из инфиксной записи в ОПН с помощью стека операторов. Скобки обрабатываются рекурсивно.
3. `FromPolsk()` вычисляет ОПН-очередь: числа кладутся в стек, при встрече оператора извлекаются два операнда, результат кладётся обратно.
